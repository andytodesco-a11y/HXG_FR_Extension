Vb.Net

How to create ESPRIT EDGE Vb.Net tutorials extension

  1. Copy the folder with Vb.Net extension source files from "C:\Program Files\Hexagon\ESPRIT EDGE\Help\1033\API Tutorial Samples" to the appropriate place on your computer.
  2. Open and build the TutorialVbNet.sln with VS2017 or VS2019 or vs2022.
  3. After the successful build, the extension will be copied to "C:\Users\Public\Documents\Hexagon\ESPRIT EDGE\Data\Extensions\Tutorial VbNet".




How to enable ESPRIT EDGE Vb.Net tutorials extension

  1. All Vb.Net tutorials mentioned in the help are installed with ESPRIT EDGE as Vb.Net extension. Check extension status through File->Extensions and enable it .
 
  1. Once enabled, you will see Tutorials tab on the ribbon with Vb.Net Tutorials under it.
 




Executing tutorial code samples

ESPRIT EDGE Vb.Net extension allows to execute the tutorials code samples. Click on Tutorial->Vb.Net Tutorials on the ribbon to see the available tutorials. To launch the code, select a tutorial in the tree and press the Execute button.
