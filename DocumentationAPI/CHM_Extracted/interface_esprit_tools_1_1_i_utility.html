espMachiningType GetToolMachine(espTechnologyType Type)
Function GetToolMachine(ByVal Type As espTechnologyType) As espMachiningType
Function GetToolMachine(ByVal Type As espTechnologyType) As espMachiningType

Returns a value to indicate the machining mode for a tool technology value. GetToolMachine returns a value of 1 for a mill type tool technology such as espToolMillBallMill and a value of 2 for a lathe type tool technology such as espToolLatheTurning. An error will be thrown for non-tool technology values (those in the espTechnologyType enumeration that begin with espTech and not espTool).
