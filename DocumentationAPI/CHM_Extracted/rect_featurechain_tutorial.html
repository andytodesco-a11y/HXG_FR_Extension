Rectangular FeatureChain

In this tutorial you will prompt the user to select two points and then create a rectangular FeatureChain with those two points representing the diagonal. The important concept introduced in this tutorial is "Get" method of the Document - it uses GetSegment to create temporary segments. Those segments are nested directly into the commands that add to the FeatureChain. Note how the coordinate properties of the chosen points are accessed.

Before running this code, create some points in ESPRIT EDGE. You will then be able to those points for the corners of the rectangle. You can also use GRID or SNAP mode while running the code instead of creating points first.

public void RectangularFeatureChain()
{
Esprit.Point point1 = null;
Esprit.Point point2 = null;
try
{
point1 = Document.GetPoint("Pick First Corner Point");
point2 = Document.GetPoint("Pick Second Corner Point");
}
catch (System.Runtime.InteropServices.COMException)
{
// Element was not set
return;
}
if (point1 == null || point2 == null)
{
return;
}
var featureChain = Document.FeatureChains.Add(point1);
featureChain.Add(Document.GetSegment(point1, Document.GetPoint(point1.X, point2.Y, 0)));
featureChain.Add(Document.GetSegment(Document.GetPoint(point1.X, point2.Y, 0), point2));
featureChain.Add(Document.GetSegment(point2, Document.GetPoint(point2.X, point1.Y, 0)));
featureChain.Add(Document.GetSegment(Document.GetPoint(point2.X, point1.Y, 0), point1));
featureChain.Color = ColorHelper.ColorToUInt(System.Drawing.Color.Blue);
Document.Refresh();
}
The following function is used to convert the color value:
public static uint ColorToUInt(System.Drawing.Color color)
{
var value = (uint)((color.A << 24) | (color.B << 16) |
(color.G << 8) | color.R);
return value;
}

Public Sub RectangularFeatureChain()
Dim point1 As Esprit.Point = Nothing
Dim point2 As Esprit.Point = Nothing
Try
point1 = Document.GetPoint("Pick First Corner Point")
point2 = Document.GetPoint("Pick Second Corner Point")
Catch ex As System.Runtime.InteropServices.COMException
' Point was not set
Return
End Try
If point1 Is Nothing OrElse point2 Is Nothing Then
Return
End If
Dim featureChain As Esprit.FeatureChain = Document.FeatureChains.Add(point1)
Call featureChain.Add(Document.GetSegment(point1, Document.GetPoint(point1.X, point2.Y, 0)))
Call featureChain.Add(Document.GetSegment(Document.GetPoint(point1.X, point2.Y, 0), point2))
Call featureChain.Add(Document.GetSegment(point2, Document.GetPoint(point2.X, point1.Y, 0)))
Call featureChain.Add(Document.GetSegment(Document.GetPoint(point2.X, point1.Y, 0), point1))
featureChain.Color = ColorToUInt(System.Drawing.Color.Blue)
Document.Refresh()
End Sub
The following function is used to convert the color value:
Public Function ColorToUInt(color As System.Drawing.Color) As UInteger
Dim a As UInteger = color.A
Dim r As UInteger = color.R
Dim g As UInteger = color.G
Dim b As UInteger = color.B
Dim resuls As UInteger = ((a << 24) Or (b << 16) Or (g << 8) Or (r << 0))
Return resuls
End Function

Sub RectangularFeatureChain()
Dim Point1 As Esprit.Point
Dim Point2 As Esprit.Point
Set Point1 = Nothing
Set Point2 = Nothing
On Error Resume Next
Set Point1 = Document.GetPoint("Pick First Corner Point")
Set Point2 = Document.GetPoint("Pick Second Corner Point")
If Point1 Is Nothing Or Point2 Is Nothing Then
Exit Sub
End If
Dim FC As Esprit.FeatureChain
Set FC = Document.FeatureChains.Add(Point1)
Call FC.Add(Document.GetSegment(Point1, Document.GetPoint(Point1.X, Point2.Y, 0)))
Call FC.Add(Document.GetSegment(Document.GetPoint(Point1.X, Point2.Y, 0), Point2))
Call FC.Add(Document.GetSegment(Point2, Document.GetPoint(Point2.X, Point1.Y, 0)))
Call FC.Add(Document.GetSegment(Document.GetPoint(Point2.X, Point1.Y, 0), Point1))
FC.Color = vbBlue
Document.Refresh
End Sub

Do a Fit in ESPRIT EDGE if necessary and then you will see the newly created feature in the graphics with the two chosen corner points on the diagonal.
