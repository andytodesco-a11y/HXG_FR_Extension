Square Corners

In this tutorial you will create four points in square pattern. The important concepts introduced in this tutorial are using the Add method to create and add objects to a collection - we are adding Point objects to the Points collection.

Notice that the Document object is always defined and that it automatically contains a Points collection. The Document.Refresh method is used to update the graphics.

C#
VB.NET
VBA
public void SquareCorners()
{
double length = 10;
if (RequestUserInput("Enter Length", "Square Length", ref length))
{
Document.Points.Add(0, 0, 0);
Document.Points.Add(0, length, 0);
Document.Points.Add(length, length, 0);
Document.Points.Add(length, 0, 0);
Document.Refresh();
}
}
Public Sub SquareCorners()
Dim length As Double = 10.0
If RequestUserInput("Enter Length", "Square Length", length) Then
Dim point As Esprit.Point
point = Document.Points.Add(0, 0, 0)
point = Document.Points.Add(0, length, 0)
point = Document.Points.Add(length, length, 0)
point = Document.Points.Add(length, 0, 0)
Document.Refresh()
End If
End Sub
Sub SquareCorners()
Dim Length As Double
Length = InputBox("Enter Length", "Square Length", 10)
Dim P As Esprit.Point
Set P = Document.Points.Add(0, 0, 0)
Set P = Document.Points.Add(0, Length, 0)
Set P = Document.Points.Add(Length, Length, 0)
Set P = Document.Points.Add(Length, 0, 0)
Document.Refresh
End Sub

Do a Fit in ESPRIT EDGE after running the code and you will see the newly created points in the graphics.
