Philosophy of ESPRIT EDGE interfaces

Objects in ESPRIT EDGE have methods, properties and events. Methods and properties are documented in the corresponding interfaces (see Functional Categories), events interfaces are separated (see Events), so each object inherits two interfaces combining one for properties and methods and another one for events. E.g. class Document inherits IDualDocument interface and _DocumentEvents interface.

ESPRIT EDGE collections

COM collection interfaces may contain an "Item" property which is accessed in a different way in C# and VBA/VB.NET

  • C#:
    • this[] or get_Item()
  • VBA/Vb.Net:
    • Item()
ESPRIT EDGE collection iteration index starts with 1.

3D graphic scene unit type

ESPRIT EDGE 3D graphics always uses document units which is either millimeter or inch. You can get current document unit via IDualDocument.SystemUnit.

Using dynamic type (C#)

To improve the experience of interoperating with COM APIs, C# has a special dynamic type. It allows to use dynamic typing and omit the casts to static types: type checking will happen in runtime. For many COM APIs it makes your code cleaner and more readable. If you compile with the -link C# compile option, the occurrences of object in COM signatures are treated as of type dynamic. Compare these two examples how you access a cell in a Microsoft Office Excel spreadsheet with the dynamic type and without the dynamic type.

Code example with a static cast:
((Excel.Range)excelApp.Cell[1, 1].Value2 = "Name";
Excel.Range range2000 = (Excel.Range)excelApp.Cells[1, 1];
Code example with a dynamic cast:
//The access to the Value property and conversion to Excel.Range are handled by the runtime type COM binder.
excelApp.Cell[1, 1].Value2 = "Name";
Excel.Range range2000 = excelApp.Cells[1, 1];

File path separator

Use Windows file separator in FilePath arguments.

Custom CL code range

Range of 9,000-13,999 is available for users to add their own custom CL codes (see Technology.AddClCode). Anything outside this range is not supported and can result in unexpected behavior.
