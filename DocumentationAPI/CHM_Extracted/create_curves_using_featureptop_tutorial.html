Creating Curves Using FeaturePtop Commands

The Tutorial: Thrown for a Curve showed how the AddFromElement method creates a curve from another graphics object. A convenient graphic object for associating points together is the FeaturePtop. We can easily create a curve from a feature ptop using AddFromElement, and this gives us the flexibility to use commands built into ESPRIT EDGE for feature ptops with curves as well.

First, let us start with a basic function to create a curve from grouped points, only this time by way of a FeaturePtop instead of having to create an array of points as in the Tutorial: Thrown for a Curve.

private void GroupedPointsToCurveViaPtop()
{
Esprit.FeaturePtop featurePtop = null;
for (var i = 1; i <= Document.Group.Count; i++)
{
var graphicObject = Document.Group[i] as Esprit.IGraphicObject;
if (graphicObject.GraphicObjectType == EspritConstants.espGraphicObjectType.espPoint)
{
var point = graphicObject as Esprit.Point;
if (featurePtop == null)
{
featurePtop = Document.FeaturePtops.Add(point);
}
else
{
featurePtop.Add(point);
}
}
}
if (featurePtop == null)
{
MessageBox.Show("There are no selected points");
return;
}
if (featurePtop.Count > 2 && MessageBox.Show("Close the curve?", "Close Curve", MessageBoxButton.YesNo) == MessageBoxResult.Yes)
{
featurePtop.Add(featurePtop.Item[1]);
}
try
{
GraphicObjectHelper.SetDefaultAttributes(Document.Curves.AddFromElement(featurePtop) as Esprit.GraphicObject, _application.Configuration);
}
catch (System.Runtime.InteropServices.COMException)
{ }
Document.FeaturePtops.Remove(featurePtop.Key);
Document.Refresh();
}

Sub GroupedPointsToCurveViaPtop()
Dim FP As Esprit.FeaturePtop
' loop through the group
Dim I As Long
For I = 1 To Document.Group.Count
If Document.Group.Item(I).GraphicObjectType = espPoint Then
' add the point to our ptop
If FP Is Nothing Then
Set FP = Document.FeaturePtops.Add(Document.Group.Item(I))
Else
Call FP.Add(Document.Group.Item(I))
End If
End If
Next
' prompt the user if they want to close the curve
If MsgBox("Close the curve?", vbYesNo, "Close Curve") = vbYes Then
' re-add the first point to the end of the ptop
Call FP.Add(FP.Item(1))
End If
' convert the ptop to a curve
On Error Resume Next
Dim Crv As Esprit.Curve
Set Crv = DefaultAttributes(Document.Curves.AddFromElement(FP))
' remove the ptop and update the graphics
Call Document.FeaturePtops.Remove(FP.Key)
Document.Refresh
End Sub

What can we do with this ability to create curves from ptops? Well, if you use the Add Box command on the Curve toolbar, one limitation that you may run into is that it will only accept the first 200 points in the box. We have already seen how to create a curve from more than 200 points. How could we replace one of the commands on the curve toolbar to run our code? Well, we first saw in Tutorial: Using Class Modules to Respond to ESPRIT EDGE Events that the Application has events such as BeforeCommandRun, to let us know when a command is selected. We can check the command ID in that event, and then run our code for the appropriate command. Besides events such as BeforeCommandRun, the Application also has methods such as RunCommand that allow us to run commands on the toolbars by their command ids.

So in BeforeCommandRun event we need to check the Auto Search Curve command that automatically creates a curve from the group. Before creating a curve execute RunCommand for Manual Ptop (id = 3254) and Cycle Stop (id = 3201) commands.

private void OnBeforeCommandRun(int nCmdId, int nAuxId, ref bool bOverride)
{
if (nCmdId != (int)EspritConstants.espCommands.espCmdNurb3dCurveAuto)
{
return;
}
var origCount = Document.FeaturePtops.Count;
_application.RunCommand(EspritConstants.espCommands.espCmdDefptopNewOpen);
_application.RunCommand(EspritConstants.espCommands.espCmdDefstructEndcycle);
if (Document.FeaturePtops.Count > origCount)
{
var lastPtop = Document.FeaturePtops[Document.FeaturePtops.Count];
try
{
GraphicObjectHelper.SetDefaultAttributes(Document.Curves.AddFromElement(lastPtop) as Esprit.GraphicObject, _application.Configuration);
}
catch (System.Runtime.InteropServices.COMException)
{ }
Document.FeaturePtops.Remove(lastPtop.Key);
bOverride = true;
}
}
Private Sub OnBeforeCommandRun(nCmdId As Integer, nAuxId As Integer, ByRef bOverride As Boolean)
If nCmdId <> CType(EspritConstants.espCommands.espCmdNurb3dCurveAuto, Integer) Then
Return
End If
Dim origCount As Integer = Document.FeaturePtops.Count
_application.RunCommand(EspritConstants.espCommands.espCmdDefptopNewOpen)
_application.RunCommand(EspritConstants.espCommands.espCmdDefstructEndcycle)
If Document.FeaturePtops.Count > origCount Then
Dim lastPtop As Esprit.FeaturePtop = Document.FeaturePtops(Document.FeaturePtops.Count)
Try
GraphicObjectHelper.SetDefaultAttributes(Document.Curves.AddFromElement(lastPtop), _application.Configuration)
Catch ex As System.Runtime.InteropServices.COMException
End Try
Document.FeaturePtops.Remove(lastPtop.Key)
bOverride = True
End If
End Sub
Dim WithEvents MyApp As Esprit.Application
Attribute MyApp.VB_VarHelpID = -1
Private Sub Class_Initialize()
Set MyApp = Application
End Sub
Private Sub MyApp_BeforeCommandRun(ByVal nCmdId As Long, ByVal nAuxId As Long, bOverride As Boolean)
If nCmdId = espCmdNurb3dCurveAuto Then ' auto search curve
Dim OriginalCount As Long
OriginalCount = Document.FeaturePtops.Count
' try making a group ptop first
Call Application.RunCommand(espCmdDefptopNewOpen) ' manual ptop
Call Application.RunCommand(espCmdDefstructEndcycle) ' cycle stop
' see if a feature ptop was created
If Document.FeaturePtops.Count > OriginalCount Then
' convert the last ptop created to a curve and remove
Dim LastPtop As Esprit.FeaturePtop
Set LastPtop = Document.FeaturePtops.Item(Document.FeaturePtops.Count)
Call DefaultAttributes(Document.Curves.AddFromElement(LastPtop))
Call Document.FeaturePtops.Remove(LastPtop.Key)
' override the auto search curve
bOverride = True
End If
End If
End Sub
Dim AutoSeachCurveClass As CreateCurvesUsingFPtopHandler
Sub ReplaceAutoSearchCurveCommand()
Set AutoSeachCurveClass = New CreateCurvesUsingFPtopHandler
End Sub

Group some points, and then click on the Auto Search command on the Curve toolbar. Before running the code, it will prompt you for the Starting Point. Press Esc to abort. Now execute the tutorial code, then click the Auto Search command icon again. This time, it will automatically create a curve through the grouped points.
