Creating Cross Section and Surface Mesh Curves

We have seen several applications for creating curves in the previous tutorials. Another common application is the need to create multiple curves such as cross sections from large collections of point data. For example, you may import some point data that was collected from a Coordinate Measure Machine (CMM). In order to create a surface, you first need to create curves through those points. Those curves might run in one direction as parallel cross sections, or they may form an ordered mesh in two directions. Depending on how the point data is organized, we can write code to generate those curves.

Let us start with cross sections. We can assume that the sections are normal (perpendicular) to either the X, Y or Z axis. For that axis, the sections start at a certain value and are spaced at a constant interval. We may want to account for a slight deviation of the points from that interval. We can easily test the desired coordinate to see what cross-section the point lies in relative to the start, and then add that point to the curve for that section.

To be flexible, let us create a function that can create the curves normal to any of the three global axes, and that accepts an enumerated value as an argument to designate the normal axis.

private static void CreateCrossSectionCurves(Esprit.Application app, EspSectionNormalAxis normalAxis, double start, double step, double deviation = 0)
{
var document = app.Document;
Dictionary<int, Esprit.FeaturePtop> ptops = new Dictionary<int, Esprit.FeaturePtop>();
Dictionary<int, double> min = new Dictionary<int, double>();
Dictionary<int, double> max = new Dictionary<int, double>();
for (var i = 1; i <= document.Group.Count; i++)
{
var graphicObject = document.Group[i] as Esprit.IGraphicObject;
if (graphicObject == null || graphicObject.GraphicObjectType != EspritConstants.espGraphicObjectType.espPoint)
{
continue;
}
var point = graphicObject as Esprit.Point;
double pointCoordinate = 0;
switch(normalAxis)
{
case EspSectionNormalAxis.espSectionNormalAxisX:
pointCoordinate = point.X;
break;
case EspSectionNormalAxis.espSectionNormalAxisY:
pointCoordinate = point.Y;
break;
case EspSectionNormalAxis.espSectionNormalAxisZ:
pointCoordinate = point.Z;
break;
}
int sectionNumber = (int)(Math.Abs(pointCoordinate - start) / step);
if (!min.ContainsKey(sectionNumber) && !max.ContainsKey(sectionNumber))
{
min.Add(sectionNumber, start + sectionNumber * step - Math.Abs(deviation));
max.Add(sectionNumber, start + sectionNumber * step + Math.Abs(deviation));
}
if (pointCoordinate >= min[sectionNumber] && pointCoordinate <= max[sectionNumber])
{
if (!ptops.ContainsKey(sectionNumber))
{
ptops.Add(sectionNumber, document.FeaturePtops.Add(point));
}
else
{
ptops[sectionNumber].Add(point);
}
}
foreach(var section in ptops.Values)
{
try
{
if (section.Count > 1)
{
GraphicObjectHelper.SetDefaultAttributes(document.Curves.AddFromElement(section) as Esprit.GraphicObject, app.Configuration);
}
document.FeaturePtops.Remove(section.Key);
}
catch(Exception)
{
}
}
}
}

Dim max As Dictionary(Of Integer, Double) = New Dictionary(Of Integer, Double)()

Continue For

pointCoordinate = point.Z

min.Add(sectionNumber, start + sectionNumber * stp - Math.Abs(deviation))
max.Add(sectionNumber, start + sectionNumber * stp + Math.Abs(deviation))

espSectionNormalAxisY = 2
espSectionNormalAxisZ = 3
End Enum
Function CreateCrossSectionCurves(NormalAxis As espSectionNormalAxis, Start As Double, Step As Double, Optional Deviation As Double) As Esprit.Curve()
Dim SectionPtop() As Esprit.FeaturePtop, SectionCount As Long
Dim SectionNumber As Long, Min() As Double, Max() As Double
' to be more effecient, only loop through the group once
' determine appropriate section from coordinate value
Dim I As Long
For I = 1 To Document.Group.Count
If Document.Group.Item(I).GraphicObjectType = espPoint Then
Dim P As Esprit.Point
Set P = Document.Group.Item(I)
' find the nearest increment of the step from the start where the point lies
Dim PointCoordinate As Double
Select Case NormalAxis
Case espSectionNormalAxisX
PointCoordinate = P.X
Case espSectionNormalAxisY
PointCoordinate = P.Y
Case espSectionNormalAxisZ
PointCoordinate = P.Z
End Select
SectionNumber = Round(Abs((PointCoordinate - Start) / Step), 0)
' redimension arrays if this section is beyond current last section
' note that the first section at the Start value is section 0
If SectionNumber > (SectionCount - 1) Then
SectionCount = SectionNumber + 1
ReDim Preserve Min(0 To SectionCount - 1) As Double
ReDim Preserve Max(0 To SectionCount - 1) As Double
ReDim Preserve SectionPtop(0 To SectionCount - 1) As Esprit.FeaturePtop
End If
' make sure the allowable minimums and maximums are set
If (Min(SectionNumber) = 0) And (Max(SectionNumber) = 0) Then
Min(SectionNumber) = Start + SectionNumber * Step - Abs(Deviation)
Max(SectionNumber) = Start + SectionNumber * Step + Abs(Deviation)
End If
' if the point lies within the range for that section, add to ptop
If (PointCoordinate >= Min(SectionNumber)) And (PointCoordinate <= Max(SectionNumber)) Then
If SectionPtop(SectionNumber) Is Nothing Then
Set SectionPtop(SectionNumber) = Document.FeaturePtops.Add(P)
Else
Call SectionPtop(SectionNumber).Add(P)
End If
End If
End If
Next
' loop through the resulting array of features and convert to curve array
Dim SectionCurves() As Esprit.Curve, CurveCount As Long
CurveCount = 0
For SectionNumber = 0 To SectionCount - 1
If Not (SectionPtop(SectionNumber) Is Nothing) Then
' since we cannot create a curve out of a single point ptop
If SectionPtop(SectionNumber).Count > 1 Then
' expand the curve array to hold the new curve and convert
CurveCount = CurveCount + 1
ReDim Preserve SectionCurves(1 To CurveCount) As Esprit.Curve
Set SectionCurves(CurveCount) = DefaultAttributes(Document.Curves.AddFromElement(SectionPtop(SectionNumber)))
End If
' remove the ptop from which the curve was converted
Call Document.FeaturePtops.Remove(SectionPtop(SectionNumber).Key)
End If
Next
' return the resulting curve array
CreateCrossSectionCurves = SectionCurves
End Function

If you do not have a file that contains a cloud of points, you could make a simple one by using the Point Array command on the geometry toolbar to create a grid of points. Examine the curves that result. You could go on to put a Lofted Surface through these curves using the Lofted Surface command on the Surfaces toolbar.

Note that in the code above we assume that the points in any given section are going to be in order, and not jump around in that section. If that is not the case then we could possibly Optimize the resulting feature ptops before converting them to curves. Optimize was covered in Tutorial: Optimizing FeaturePtop Objects. These functions also generate open sections, such as for a surface, but the first point could be added at the end to close each feature such as for a solid as well.

The important thing is to look at your data, and write code appropriate to how it is organized.

For example, maybe we do not need to be concerned with the coordinates of the points. Suppose we are given a mesh of points such that the points are in order. The points go along a certain direction, so we know when we come to a point that lies in the opposite direction, it is the start of a new curve. After we get the curves in that first direction, we can simply take the first point in each curve as being in the first curve in the second direction, the second point in each of the curves in the first direction as being in the second curve in the second direction, and so on.

private void CreateXYMesh()
{
if (Document.Group.Count == 0)
{
System.Windows.MessageBox.Show("Please group points before running this function.", "CrossSectionSurfaceMeshCurvesTutorial", MessageBoxButton.OK);
return;
}
Esprit.Point prevPoint = null;
Esprit.Point currentPoint = null;
List<Esprit.FeaturePtop> featurePtops = new List<Esprit.FeaturePtop>();
for (var i = 1; i <= Document.Group.Count; i++)
{
var graphicObject = Document.Group[i] as Esprit.IGraphicObject;
if (graphicObject == null || graphicObject.GraphicObjectType != EspritConstants.espGraphicObjectType.espPoint)
{
continue;
}
string primaryDirection = string.Empty;
var point = graphicObject as Esprit.Point;
if (prevPoint == null)
{
prevPoint = point;
featurePtops.Add(Document.FeaturePtops.Add(prevPoint));
}
else
{
if (currentPoint == null)
{
currentPoint = point;
var deltaX = currentPoint.X - prevPoint.X;
var deltaY = currentPoint.Y - prevPoint.Y;
if (Math.Abs(deltaX) > Math.Abs(deltaY))
{
primaryDirection = (deltaX > 0)
? "+X"
: "-X";
}
else
{
primaryDirection = (deltaY > 0)
? "+Y"
: "-Y";
}
}
else
{
prevPoint = currentPoint;
currentPoint = point;
}
if (currentPoint != null && prevPoint != null)
{
bool isDirectionChanged = false;
switch (primaryDirection)
{
case "+X": if (currentPoint.X < prevPoint.X) isDirectionChanged = true; break;
case "-X": if (currentPoint.X > prevPoint.X) isDirectionChanged = true; break;
case "+Y": if (currentPoint.Y < prevPoint.Y) isDirectionChanged = true; break;
case "-Y": if (currentPoint.Y > prevPoint.Y) isDirectionChanged = true; break;
}
if (isDirectionChanged)
{
featurePtops.Add(Document.FeaturePtops.Add(currentPoint));
}
else
{
featurePtops[featurePtops.Count-1].Add(currentPoint);
}
}
}
}
var maxCount = 0;
for (var i = 0; i < featurePtops.Count; i++)
{
if (maxCount < featurePtops[i].Count)
{
maxCount = featurePtops[i].Count;
}
}
var featurePtops2 = new Esprit.FeaturePtop[maxCount];
for (var i = 0; i < maxCount; i++)
{
if (i < featurePtops[0].Count)
{
featurePtops2[i] = Document.FeaturePtops.Add(featurePtops[0].Item[i+1] as Esprit.Point);
}
else
{
featurePtops2[i] = Document.FeaturePtops.Add(featurePtops[0].Item[featurePtops[0].Count] as Esprit.Point);
}
for (var j = 1; j < featurePtops.Count; j++)
{
if (i < featurePtops[j].Count)
{
featurePtops2[i].Add(featurePtops2[j].Item[i + 1]);
}
else
{
featurePtops2[i].Add(featurePtops2[j].Item[featurePtops[0].Count]);
}
}
}
for (var i = 0; i < featurePtops.Count; i++)
{
try
{
GraphicObjectHelper.SetDefaultAttributes(Document.Curves.AddFromElement(featurePtops[i]) as Esprit.GraphicObject, _application.Configuration);
Document.FeaturePtops.Remove(featurePtops[i].Key);
}
catch(Exception)
{
_application.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeWarning, "CrossSectionSurfaceMeshCurvesTutorial", $"Creating curve [{i}] from featurePtop failed");
}
}
for (var i = 0; i < featurePtops2.Length; i++)
{
try
{
GraphicObjectHelper.SetDefaultAttributes(Document.Curves.AddFromElement(featurePtops2[i]) as Esprit.GraphicObject, _application.Configuration);
Document.FeaturePtops.Remove(featurePtops2[i].Key);
}
catch (Exception)
{
_application.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeWarning, "CrossSectionSurfaceMeshCurvesTutorial", $"Creating curve [{i}] from featurePtop failed");
}
}
Document.Refresh();
}

If Math.Abs(deltaX) > Math.Abs(deltaY) Then
primaryDirection = IIf(deltaX > 0, "+X", "-X")
Else
primaryDirection = IIf(deltaY > 0, "+Y", "-Y")
End If
Else
prevPoint = currentPoint
currentPoint = point
End If
If Not currentPoint Is Nothing AndAlso Not prevPoint Is Nothing Then
Dim isDirectionChanged As Boolean = False
Select Case primaryDirection
Case "+X" : If (currentPoint.X < prevPoint.X) Then isDirectionChanged = True
Case "-X" : If (currentPoint.X > prevPoint.X) Then isDirectionChanged = True
Case "+Y" : If (currentPoint.Y < prevPoint.Y) Then isDirectionChanged = True
Case "-Y" : If (currentPoint.Y > prevPoint.Y) Then isDirectionChanged = True
End Select
If isDirectionChanged Then
featurePtops.Add(Document.FeaturePtops.Add(currentPoint))
Else
featurePtops(featurePtops.Count - 1).Add(currentPoint)
End If
End If
End If
Next
Dim maxCount As Integer = 0
For i As Integer = 0 To featurePtops.Count - 1
If maxCount < featurePtops(i).Count Then
maxCount = featurePtops(i).Count
End If
Next
Dim featurePtops2(maxCount - 1) As Esprit.FeaturePtop
For i As Integer = 0 To maxCount - 1
If i < featurePtops(0).Count Then
featurePtops2(i) = Document.FeaturePtops.Add(featurePtops(0).Item(i + 1))
Else
featurePtops2(i) = Document.FeaturePtops.Add(featurePtops(0).Item(featurePtops(0).Count))
End If
For j As Integer = 1 To featurePtops.Count - 1
If i < featurePtops(j).Count Then
featurePtops2(i).Add(featurePtops2(j).Item(i + 1))
Else
featurePtops2(i).Add(featurePtops2(j).Item(featurePtops(0).Count))
End If
Next
Next
For i As Integer = 0 To featurePtops.Count - 1
Try
GraphicObjectHelper.SetDefaultAttributes(Document.Curves.AddFromElement(featurePtops(i)), _application.Configuration)
Document.FeaturePtops.Remove(featurePtops(i).Key)
Catch ex As Exception
_application.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeWarning, "CrossSectionSurfaceMeshCurvesTutorial", $"Creating curve [{i}] from featurePtop failed")
End Try
Next
For i As Integer = 0 To featurePtops2.Length - 1
Try
GraphicObjectHelper.SetDefaultAttributes(CType(Document.Curves.AddFromElement(featurePtops2(i)), Esprit.GraphicObject), _application.Configuration)
Document.FeaturePtops.Remove(featurePtops2(i).Key)
Catch ex As Exception
_application.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeWarning, "CrossSectionSurfaceMeshCurvesTutorial", $"Creating curve [{i}] from featurePtop failed")
End Try
Next
Document.Refresh()
End Sub
Sub CreateXYMesh()
If Document.Group.Count = 0 Then
Call MsgBox("Please group points before running this macro.", vbOKOnly, "Error")
Exit Sub
End If
' loop through all of the points on the group
Dim PreviousPoint As Esprit.Point, CurrentPoint As Esprit.Point
Dim FP1() As Esprit.FeaturePtop, FP1Count As Long
Dim I As Long
For I = 1 To Document.Group.Count
If Document.Group.Item(I).GraphicObjectType = espPoint Then
' compare this point with the previous point
Dim DeltaX As Double, DeltaY As Double, PrimaryDirection As String
PrimaryDirection = ""
If PreviousPoint Is Nothing Then
Set PreviousPoint = Document.Group.Item(I)
FP1Count = 1
ReDim FP1(FP1Count) As Esprit.FeaturePtop
Set FP1(FP1Count) = Document.FeaturePtops.Add(PreviousPoint)
Else
If CurrentPoint Is Nothing Then
' determine primary direction of the mesh from the first two points
Set CurrentPoint = Document.Group.Item(I)
DeltaX = CurrentPoint.X - PreviousPoint.X
DeltaY = CurrentPoint.Y - PreviousPoint.Y
If Abs(DeltaX) > Abs(DeltaY) Then
If DeltaX > 0 Then
PrimaryDirection = "+X"
Else
PrimaryDirection = "-X"
End If
Else
If DeltaY > 0 Then
PrimaryDirection = "+Y"
Else
PrimaryDirection = "-Y"
End If
End If
Else
Set PreviousPoint = CurrentPoint
Set CurrentPoint = Document.Group.Item(I)
End If
' see if this point continues to go in the primary direction
Dim DirectionChange As Boolean
DirectionChange = False
Select Case PrimaryDirection
Case "+X"
If CurrentPoint.X < PreviousPoint.X Then DirectionChange = True
Case "-X"
If CurrentPoint.X > PreviousPoint.X Then DirectionChange = True
Case "+Y"
If CurrentPoint.Y < PreviousPoint.Y Then DirectionChange = True
Case "-Y"
If CurrentPoint.Y > PreviousPoint.Y Then DirectionChange = True
End Select
If DirectionChange Then
' start a new ptop
FP1Count = FP1Count + 1
ReDim Preserve FP1(FP1Count) As Esprit.FeaturePtop
Set FP1(FP1Count) = Document.FeaturePtops.Add(CurrentPoint)
Else
' add to last ptop
Call FP1(FP1Count).Add(CurrentPoint)
End If
End If
End If
Next
' now loop back through the primary ptop array
' determine the maximum number of points in the primary direction
Dim FP2() As Esprit.FeaturePtop, FP2Count As Long
For I = 1 To FP1Count
If FP1(I).Count > FP2Count Then
FP2Count = FP1(I).Count
End If
Next
ReDim FP2(FP2Count) As Esprit.FeaturePtop
' now create the secondard array from the items in the primary
For I = 1 To FP2Count
' start with the corresponding point from the first primary ptop
If I <= FP1(1).Count Then
Set FP2(I) = Document.FeaturePtops.Add(FP1(1).Item(I))
Else
' use the last point if we run out
Set FP2(I) = Document.FeaturePtops.Add(FP1(1).Item(FP1(1).Count))
End If
' loop through the remaining primary ptops
Dim J As Long
For J = 2 To FP1Count
If I <= FP1(J).Count Then
Call FP2(I).Add(FP1(J).Item(I))
Else
' use the last point if we run out
Call FP2(I).Add(FP1(J).Item(FP1(1).Count))
End If
Next
Next
On Error Resume Next
' now convert the ptops to curves and remove
For I = 1 To FP1Count
Call DefaultAttributes(Document.Curves.AddFromElement(FP1(I)))
Call Document.FeaturePtops.Remove(FP1(I).Key)
Next
For I = 1 To FP2Count
Call DefaultAttributes(Document.Curves.AddFromElement(FP2(I)))
Call Document.FeaturePtops.Remove(FP2(I).Key)
Next
Document.Refresh
End Sub

Again, if you do not have a file that contains a cloud of points, you could make a simple one by using the Point Array command on the geometry toolbar to create several lines of points. Each line should have the same number of points and should go in the same primary direction, but their angle and distance between points could vary. Then examine the mesh of curves that result. You could go on to put a surface through these curves using the Curve Mesh Surface command on the Surfaces toolbar.
