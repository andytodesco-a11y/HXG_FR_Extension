Swarf Cycle

This tutorial demonstrates the usage of Swarf operation.

Before running the tutorial, load "Help\1033\API Tutorial Samples\Data\SwarfCycle\SwarfCycle.esprit". This file already has an impeller and operations for the turning of the OD and the roughing of the vane. We need to create two operations to cut two sides: left blade and right blade. For each side the top and bottom chain features and the related FreeForm Features have been created. The FFF contains in its Part the side face to cut and in its Check the faces not to be touched for each case.

private EspritTechnology.TechMill5xSwarfMerged CreateSwarfCycle(string operationName)
{

SwarfStartFromOffsetMorph = EspritConstants.espMoldSwarfStartFromOffsetMorph.espMoldSwarfStartFromOffsetMorphNo
};

{

};
swarfCycle.set_EntryMovesStrategies(entries.ToArray());
return swarfCycle;
}
private void AddSwarfOperation(EspritTechnology.TechMill5xSwarfMerged technology, Esprit.IGraphicObject feature)
{
Document.Group.Clear();
Document.Group.Add(feature);
Document.PartOperations.Add(technology, feature);
Document.Group.Clear();
}
private void SwarfLeft()
{
EspritTechnology.TechMill5xSwarfMerged swarfCycle = CreateSwarfCycle("Swarf Left");
swarfCycle.UpperProfile = "6,6"; // identifies "LeftBlade_Top"
swarfCycle.LowerProfile = "6,7"; // identifies "LeftBlade_Bottom"
foreach (Esprit.FreeFormFeature feature in Document.FreeFormFeatures)
{
if (feature.Name == "LeftBlade")
{
AddSwarfOperation(swarfCycle, feature);
break;
}
}
}
private void SwarfRight()
{
EspritTechnology.TechMill5xSwarfMerged swarfCycle = CreateSwarfCycle("Swarf Right");
swarfCycle.UpperProfile = "6,8"; // identifies "RightBlade_Top"
swarfCycle.LowerProfile = "6,9"; // identifies "RightBlade_Bottom"
foreach (Esprit.FreeFormFeature feature in Document.FreeFormFeatures)
{
if (feature.Name == "RightBlade")
{
AddSwarfOperation(swarfCycle, feature);
break;
}
}
}

For the Swarf Left operation in the Event Window you'll get the Swarf Report:

Operation: Swarf Left Max Violation of Target: 0.007 Max Distance from Target 0.710

This means that, in the worst points during this cut, the tool violates the target by 0.007 mm and it stays away 0.710 mm with respect to the model target.

For the Swarf Right operation in the Event Window you'll get the Swarf Report:

Operation: Swarf Right Max Violation of Target: 0.000 Max Distance from Target 0.787

You also may select Swarf Left or Swarf Right and run the Toolpath Analysis (Analysis -> Toolpath on the Ribbon).
