More Configuration: Grid and Background

Tutorial: Introduction to the Configuration Object showed how to use attribute settings from the Configuration object, but there are many more properties available in the Configuration. For example, settings for Importing and Exporting files are found in such objects as IgesFile and ParasolidFile. Dimensioning Parameters are in ConfigurationDimension and settings for the Background are in ConfigurationBackground. General options such as properties for grid parameters and the showing of system axes are also found in the Configuration.

The tutorial will cover a couple of examples.

private void ScaleGrid()
{
var scaleFactor = 1.0;
if (RequestUserInput("Enter Grid Scale Factor", "Scale Factor", ref scaleFactor))
{
_application.Configuration.EnableGridMode = true;
_application.Configuration.GridAngle *= scaleFactor;
if (_application.Document.SystemUnit == EspritConstants.espUnitType.espInch)
{
_application.Configuration.GridDxInch *= scaleFactor;
_application.Configuration.GridDyInch *= scaleFactor;
_application.Configuration.GridDzInch *= scaleFactor;
_application.Configuration.GridRadiusInch *= scaleFactor;
}
if (_application.Document.SystemUnit == EspritConstants.espUnitType.espMetric)
{
_application.Configuration.GridDxMetric *= scaleFactor;
_application.Configuration.GridDyMetric *= scaleFactor;
_application.Configuration.GridDzMetric *= scaleFactor;
_application.Configuration.GridRadiusMetric *= scaleFactor;
}
}
}

Sub ScaleGrid()
Dim ScaleFactor As Double
ScaleFactor = Abs(Val(InputBox("Enter Grid Scale Factor", "Scale Factor", 1)))
With Application.Configuration
.EnableGridMode = True
.GridAngle = .GridAngle * ScaleFactor
If Document.SystemUnit = espMetric Then
.GridDxMetric = .GridDxMetric * ScaleFactor
.GridDyMetric = .GridDyMetric * ScaleFactor
.GridDzMetric = .GridDzMetric * ScaleFactor
.GridRadiusMetric = .GridRadiusMetric * ScaleFactor
Else
.GridDxInch = .GridDxInch * ScaleFactor
.GridDyInch = .GridDyInch * ScaleFactor
.GridDzInch = .GridDzInch * ScaleFactor
.GridRadiusInch = .GridRadiusInch * ScaleFactor
End If
End With
End Sub

Public Function ColorToUInt(color As System.Drawing.Color) As UInteger
Dim a As UInteger = color.A
Dim r As UInteger = color.R
Dim g As UInteger = color.G
Dim b As UInteger = color.B
Dim resuls As UInteger = ((a << 24) Or (b << 16) Or (g << 8) Or (r << 0))
Return resuls
End Function

Note the use of both the _Change and _Scroll events for the scroll bars. _Change responds when the scroll bar is clicked (and thus changed by the LargeChange value). _Scroll responds when the button is dragged (and thus changed by the SmallChange value).
