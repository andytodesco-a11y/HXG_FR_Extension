Function Close() As espCloseFeatureChainError

Close open chain feature.

Returns
Returns error ID (espCloseFeatureChainError) if something wrong
Remarks
Close an open feature chain.
If there is/are intersection point(s) between the open feature and the line from the feature
start to end points, the intersection points that are sorted along the line must be monotonic
increasing along the feature. otherwise the result may not be expected.
