Naming And Recalling Groups

This tutorial shows how to save a group with a name so that it can be recalled later. This can be done via the SelectionSet object. A SelectionSet is a named collection of GraphicObject objects. The SelectionSets collection is then a collection of all SelectionSet objects in the document.

public void NameGroup()
{
var groupName = $"Group {Document.SelectionSets.Count + 1}";
if (!RequestUserInput("Enter Group Name", $"Group Name:", ref groupName) || groupName.Length == 0)
{
return;
}
Esprit.SelectionSet selection = Document.SelectionSets.Add(groupName);
for (int i = 1; i <= Document.Group.Count; i++)
{
var item = Document.Group[i] as Esprit.GraphicObject;
selection.Add(item);
}
}
public void RecallGroup()
{
if (_listGroups.SelectedItems.Count > 0)
{
var selectedItem = _listGroups.SelectedItems;
string groupName = selectedItem[0] as string;
Document.Group.Clear();
var group = Document.SelectionSets[groupName];
for (int i = 1; i <= group.Count; i++)
{
var obj = group[i] as Esprit.GraphicObject;
obj.Grouped = true;
}
}
else
{
MessageBox.Show("Group Name Not Selected", "Error");
}
Document.Refresh();
}
Before running the Recall procedure, fill in the List box with the names of all existing groups:
for (int i = 1; i <= Document.SelectionSets.Count; i++)
{
string groupName = Document.SelectionSets[i].Name;
_listGroups.Items.Add(groupName);
}
Sub NameGroup()
Dim groupName As String = "Group" & Document.SelectionSets.Count + 1
If Not RequestUserInput("Enter Group Name", "Group Name", groupName) OrElse groupName.Length = 0 Then
Return
End If
Dim selectionSet As Esprit.SelectionSet = Document.SelectionSets.Add(groupName)
' loop through the Group and add grouped items to the SelectionSet
For i As Integer = 1 To Document.Group.Count
Call selectionSet.Add(Document.Group.Item(i))
Next
End Sub
Private Sub RecallGroup()
If _listGroups.SelectedItems.Count > 0 Then
Dim selectedItem = _listGroups.SelectedItems
Dim groupName As String = selectedItem(0)
' clear the current group and replace with items from the SelectionSet
Document.Group.Clear()
For j As Integer = 1 To Document.SelectionSets.Item(groupName).Count
Document.SelectionSets.Item(groupName).Item(j).Grouped = True
Next
Else
MsgBox("Group Name Not Selected", "Error")
End If
Document.Refresh()
End Sub
Before running the Recall procedure, fill in the List box with the names of all existing groups:
For i As Integer = 1 To Document.SelectionSets.Count
Dim groupName As String = Document.SelectionSets.Item(i).Name
_listGroups.Items.Add(groupName)
Next
Sub NameGroup()
Dim GroupName As String
GroupName = InputBox("Enter Group Name", "Group Name", "Group" & (Document.SelectionSets.Count + 1))
Dim SS As Esprit.SelectionSet
Set SS = Document.SelectionSets.Add(GroupName)
' loop through the Group and add grouped items to the SelectionSet
Dim I As Long
For I = 1 To Document.Group.Count
Call SS.Add(Document.Group.Item(I))
Next
End Sub
Private Sub RecallGroup()
Dim I As Long
For I = 0 To ListBoxGroups.ListCount - 1
If ListBoxGroups.Selected(I) Then
Dim GroupName As String
GroupName = ListBoxGroups.List(I)
' clear the current group and replace with items from the SelectionSet
Document.Group.Clear
Dim J As Long
For J = 1 To Document.SelectionSets.Item(GroupName).Count
Document.SelectionSets.Item(GroupName).Item(J).Grouped = True
Next
End If
Next
Document.Refresh
End Sub
Before running the Recall procedure, fill in the List box with the names of all existing groups:
For I = 1 To Document.SelectionSets.Count
GroupName = Document.SelectionSets.Item(I).Name
ListBoxGroups.AddItem (GroupName)
Next

Before runing the code open an existing document file or create some graphic objects in a new file, then group some of those items. Then execute the NameGroup procedure. Make modifications to the group and then run the NameGroup code again. Repeat this process until you have several named groups. Now run the second RecallGroup code. Select the name of one of the groups from the ListBox. You should notice the group in the graphics change to display the chosen group. Again, repeat this process, selecting other group names.
