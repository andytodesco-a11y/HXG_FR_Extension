Copy ESPRIT EDGE extension template to VS template folder

  1. Browse to "C:\Program Files\Hexagon\ESPRIT EDGE\Data\VSTemplates" and copy the entire folder "ProjectTemplates".

     

  2. Browse to "C:\Users\xxxxxxx\Documents\Visual Studio 20xx\Templates" and paste the previous copied folder.

     




Run ESPRIT EDGE extension on VS2015 - VS2017

  1. Launch your Visual Studio.
  2. Run the command: "File/New/Project".

  3. You will find under "Installed/Visual Basic" a folder called "ESPRIT EDGE". This is the folder where you can find ESPRIT EDGE templates.

     
  4. Pick the one you want, adjust folders and click on Ok to validate the project creation.
  5. Press F5 to compile and run ESPRIT EDGE. The extension will be compiled and moved to "C:\Users\Public\Documents\Hexagon\ESPRIT EDGE\Data\Extensions".
  6. To verify that the extension is properly build and loaded, go to "File/Extension" in ESPRIT EDGE.




Run ESPRIT EDGE extension on VS2019 - VS2022

  1. Launch your Visual Studio.
  2. Run the command: "File/New/Project".

  3. Set language filter "Visual Basic" and project type filter "extension".

     
  4. Pick the one ESPRIT EDGE extension type you want, click on Next to set and validate the project creation.
  5. Press F5 to compile and run ESPRIT EDGE. The extension will be compiled and moved to "C:\Users\Public\Documents\Hexagon\ESPRIT EDGE\Data\Extensions".
  6. To verify that the extension is properly build and loaded, go to "File/Extension" in ESPRIT EDGE.
