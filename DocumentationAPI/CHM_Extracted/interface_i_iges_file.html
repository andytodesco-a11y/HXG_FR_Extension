◆ SmartDrawing

bool IIgesFile::SmartDrawing
getset
C#
VB.NET
VBA
bool SmartDrawing {get; set;}
Property SmartDrawing As Boolean
Property SmartDrawing As Boolean

Sets or returns whether to automatically detect if a drawing is a 3D model or 2D layout. With SmartDrawing True, if drawing layout (IGES type 404) elements exist, the IgesFile is automatically brought in as a 2D drawing. With SmartDrawing False, then whether the file is brought in as 2D or 3D is determined by the DrawingMode.
