IComFeature Split(IComFeature Feature, double Distance)
Function Split(ByVal Feature As IComFeature, ByVal Distance As Double) As IComFeature
Function Split(ByVal Feature As IComFeature, ByVal Distance As Double) As IComFeature

Split for FeatureChain splits an element in a FeatureChain at a specified point; this allows for certain things to be specified at that point, such as the insertion of technology on an Operation applied to that FeatureChain. Split for SimuStock splits a simulation stock (SimuStock) into two or more separate solids wherever those solids have become disconnected due to material removal. Those solids could then be compared and any solids that represent slugs or waste material removed from the simulation such that only the desired rest material remains.

Parameters
[in]FeatureThe Feature value
[in]DistanceThe Distance value
Returns
