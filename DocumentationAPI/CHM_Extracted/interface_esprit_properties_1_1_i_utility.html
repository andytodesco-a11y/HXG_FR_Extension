Function CreateProperties(ByVal Document As Object, ByVal Compat2003 As Boolean, ByVal Parent As Object) As Object

Creates a CustomProperties collection attached to the given parent object. This allows a CustomProperties collection to be created on and attached to any parent object, instead of restricting it to those objects that automatically have a CustomProperties collection built in.
