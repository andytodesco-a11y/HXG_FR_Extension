◆ ProcessExpressionOnEdit

bool ITechnology::ProcessExpressionOnEdit
getset
C#
VB.NET
VBA
bool ProcessExpressionOnEdit {get; set;}
Property ProcessExpressionOnEdit As Boolean
Property ProcessExpressionOnEdit As Boolean

Sets or returns whether or not to process Parameter expressions during the editing of Technology.

Remarks
This property is necessary because of changes that Microsoft made to VBA 6.4 in regards to modal dialog boxes. When a model dialog box is displayed, such as with the Edit method of the TechnologyUtility, or with the EditTool or EditTechnology methods, then VBA will return an error to the KBM and the KBM will not be able to process the Expression. If ProcessExpressionOnEdit is False, then this error is ignored and the new value of the Parameter is passed through. If ProcessExpressionOnEdit is True, however (the default), then the KBM may reset the Parameter to the previous value due to the error from VBA. This is not a problem in separate add-in or executable projects; it only occurs when running macros from VBA projects.
