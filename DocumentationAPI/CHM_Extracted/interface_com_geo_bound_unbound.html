Function Unbound() As Object
Function Unbound() As Object

Returns the unbound (untrimmed) object that corresponds to the given bounded (ComGeoBounded) object. In the case of ComArc object, a ComCircle is returned. In the case of a ComSegment object, a ComLine is returned.

Returns
For Arc returns ComCircle. For Ellipse returns ComEllipse. For Segment returns ComSegment.
