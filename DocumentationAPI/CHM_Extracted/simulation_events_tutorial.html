Simulation Events

Event procedures are predefined for certain objects and are triggered when something happens to or with that object. The code inside the event procedure is what responds to the occurrence of that event.

The Tutorial demostrates the Simulation object events.

Define a class and add the event procedures:

VB.NET
VBA
class SimulationEventsHandler
{
private readonly Esprit.Application _application;
private readonly Esprit.Simulation _simulationObject;
private readonly string _messageBoxTitle;
public SimulationEventsHandler(Esprit.Application app)
{
_application = app;
_messageBoxTitle = "SimulationEventsTutorial";
_simulationObject = _application.Document.Simulation;
_simulationObject.StopCode = true;
_simulationObject.CollisionDetection = true;
_application.EventWindow.Visible = true;
_application.EventWindow.Clear();
_simulationObject.OnSimulationBegin += OnSimulationBegin;
_simulationObject.OnSimulationEnd += OnSimulationEnd;
_simulationObject.OnToolChangeBegin += OnToolChangeBegin;
_simulationObject.OnToolChangeEnd += OnToolChangeEnd;
_simulationObject.OnLinkBegin += OnLinkBegin;
_simulationObject.OnLinkEnd += OnLinkEnd;
_simulationObject.OnMachineOperationBegin += OnMachineOperationBegin;
_simulationObject.OnMachineOperationEnd += OnMachineOperationEnd;
_simulationObject.OnSimulationStopCode+= OnSimulationStopCode;
_simulationObject.OnSyncNodeBegin+= OnSyncNodeBegin;
_simulationObject.OnSyncNodeEnd+= OnSyncNodeEnd;
_simulationObject.OnCollision+= OnCollision;
_simulationObject.OnError+= OnError;
_simulationObject.OnWorkOffsetChanged += OnWorkOffsetChanged;
}

Now we can use this class in the following code:

C#
VB.NET
VBA
private SimulationEventsHandler _simulationEventHandler;
public void SimulationEventExample()
{
if (_simulationEventHandler == null)
{
_simulationEventHandler = new SimulationEventsHandler(_application);
}
}
Private _simulationEventHandler As SimulationEventsHandler
Public Sub SimulationEventExample()
If _simulationEventHandler Is Nothing Then
_simulationEventHandler = New SimulationEventsHandler(_application)
End If
End Sub
Dim SimulationEventTest As SimulationEventsHandler
Sub SimulationEventExample()
Set SimulationEventTest = New SimulationEventsHandler
End Sub

Before running the tutorial, create new or open a file that has several operations on it. Than start simulation, information about occurence of each SimulationEvent is logged in EventWindow.
