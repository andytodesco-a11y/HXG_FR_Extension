Extracting Feature Information

This tutorial shows how to extract information from a FeaturePtop object. Similar to collections, feature objects such as FeatureChain and FeaturePtop have a Count property and an Item. The Item for a FeaturePtop directly returns a Point object. The Count can be used to set up a loop then, and information can be gathered on each point.

The code below extracts the points from a chosen FeaturePtop, and finds the shortest and longest distances between two points in a FeaturePtop.

public void FindFeaturePtopInfo()
{
_application.EventWindow.Clear();
_application.EventWindow.Visible = true;
var shortestDistance = 0.0;
var longestDistance = 0.0;
Esprit.FeaturePtop featurePtop = null;
try
{
featurePtop = Document.GetAnyElement("Pick Feature Ptop", EspritConstants.espGraphicObjectType.espFeaturePtop) as Esprit.FeaturePtop;
}
catch(System.Runtime.InteropServices.COMException)
{
// Element was not set
return;
}
if (featurePtop == null)
{
return;
}
for (var i = 1; i <= featurePtop.Count - 1; i++)
{
var distance = Geometry.Distance(featurePtop.Item[i] as Esprit.Point, featurePtop.Item[i + 1] as Esprit.Point);
if (i == 1)
{
shortestDistance = distance;
longestDistance = distance;
}
else
{
if (distance < shortestDistance)
{
shortestDistance = distance;
}
if (distance > longestDistance)
{
longestDistance = distance;
}
}
_application.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ExtractFeatureInfoTutorial", $"The distance between items {i} and {i + 1} is {Math.Round(distance, 5)}");
}
_application.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ExtractFeatureInfoTutorial", $"The shortest distance is {Math.Round(shortestDistance, 5)}");
_application.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ExtractFeatureInfoTutorial", $"The longest distance is {Math.Round(longestDistance, 5)}");
}
The following function is used to calculate distance between two points:
public static double Distance(Point point1, Point point2 = null)
{
var x2 = point2?.X ?? 0;
var y2 = point2?.Y ?? 0;
var z2 = point2?.Z ?? 0;
return Distance(point1.X, point1.Y, point1.Z, x2, y2, z2);
}
public static double Distance(double x1, double y1, double z1, double x2, double y2, double z2)
{
var dx = x2 - x1;
var dy = y2 - y1;
var dz = z2 - z1;
return Math.Sqrt(dx * dx + dy * dy + dz * dz);
}

The following function is used to calculate distance between two points:
Public Function Distance(Point1 As Esprit.Point, Point2 As Esprit.Point) As Double
Dim DeltaX As Double, DeltaY As Double, DeltaZ As Double
DeltaX = Point2.X - Point1.X
DeltaY = Point2.Y - Point1.Y
DeltaZ = Point2.Z - Point1.Z
Distance = Math.Sqrt(DeltaX ^ 2 + DeltaY ^ 2 + DeltaZ ^ 2)
End Function
Sub FindFeaturePtopInfo()
Dim FP As Esprit.FeaturePtop
On Error Resume Next
Set FP = Document.GetAnyElement("Pick Feature Ptop", espFeaturePtop)
If FP Is Nothing Then Exit Sub ' in case user presses escape
Dim ShortestDistance As Double, LongestDistance As Double
ShortestDistance = 0
LongestDistance = 0
Dim PointNumber As Long, Dist As Double
Dist = 0
With Application.EventWindow
.Clear
.Visible = True
' loop through the points in the chosen feature ptop
For PointNumber = 1 To FP.Count - 1
' check the distance between this point and the next one
Dist = Distance(FP.Item(PointNumber), FP.Item(PointNumber + 1))
If PointNumber = 1 Then
ShortestDistance = Dist
LongestDistance = Dist
Else
If Dist < ShortestDistance Then
ShortestDistance = Dist
End If
If Dist > LongestDistance Then
LongestDistance = Dist
End If
End If
Call .AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ExtractFeatureInfoTutorial", "The distance between items " & PointNumber & " and " & (PointNumber + 1) & " is " & Round(Dist, 5))
Next
Call .AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ExtractFeatureInfoTutorial", "The shortest distance is " & Round(ShortestDistance, 5))
Call .AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ExtractFeatureInfoTutorial", "The longest distance is " & Round(LongestDistance, 5))
End With
End Sub
The following function is used to calculate distance between two points:
Public Function Distance(Point1 As Esprit.Point, Point2 As Esprit.Point) As Double
Dim DeltaX As Double, DeltaY As Double, DeltaZ As Double
DeltaX = Point2.X - Point1.X
DeltaY = Point2.Y - Point1.Y
DeltaZ = Point2.Z - Point1.Z
Distance = Sqr(DeltaX ^ 2 + DeltaY ^ 2 + DeltaZ ^ 2)
End Function

Before runing the code make sure you have one or more FeaturePtops in your ESPRIT EDGE file. Pick a FeaturePtop and the information for it will be displayed in the Output Window. If you want, you can confirm the information by smashing the FeaturePtop and placing dimensions between the points.
