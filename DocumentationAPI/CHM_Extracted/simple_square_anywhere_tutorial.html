Simple Square Anywhere

In this tutorial you will prompt the user to select a point and then create four segments to form a square with the chosen point as one of the corners. The important concept introduced in this tutorial is the power of the "Get" methods of the Document - GetPoint is not only used to create temporary points, as was done in the Simple Square Tutorial, but by passing it a prompt as an argument it is also used to have the user select a point from the graphics. The Document.GetPoint command is directly passed as an argument to the Document.Segments.Add command.

Before running this code, create some points in ESPRIT EDGE. You will then be able to pick those points as the corner of the square. You can also use GRID or SNAP mode while running the code instead of creating points first. Be sure to look for the prompt to pick the corner point in the lower left corner of the ESPRIT EDGE window.

public void SimpleSquareAnywhere()
{
var length = 10.0;
if (RequestUserInput("Enter Length", "Square Length", ref length))
{
Esprit.Point point = null;
try
{
point = Document.GetPoint("Pick Corner Point for Square");
}
catch (System.Runtime.InteropServices.COMException)
{
// Point was not set
return;
}
if (point == null)
{
return;
}
Document.Segments.Add(point, Document.GetPoint(point.X, point.Y + length, 0));
Document.Segments.Add(Document.GetPoint(point.X, point.Y + length, 0), Document.GetPoint(point.X + length, point.Y + length, 0));
Document.Segments.Add(Document.GetPoint(point.X + length, point.Y + length, 0), Document.GetPoint(point.X + length, point.Y, 0));
Document.Segments.Add(Document.GetPoint(point.X + length, point.Y, 0), point);
Document.Refresh();
}
}
Public Sub SimpleSquareAnywhere()
Dim length As Double = 10.0
If Not RequestUserInput("Enter Length", "Square Length", length) Then
Return
End If
Dim point As Esprit.Point = Nothing
Try
point = Document.GetPoint("Pick Corner Point for Square")
Catch ex As System.Runtime.InteropServices.COMException
' Point was not set
Return
End Try
If point Is Nothing Then
Return
End If
Document.Segments.Add(point, Document.GetPoint(point.X, point.Y + length, 0))
Document.Segments.Add(Document.GetPoint(point.X, point.Y + length, 0), Document.GetPoint(point.X + length, point.Y + length, 0))
Document.Segments.Add(Document.GetPoint(point.X + length, point.Y + length, 0), Document.GetPoint(point.X + length, point.Y, 0))
Document.Segments.Add(Document.GetPoint(point.X + length, point.Y, 0), point)
Document.Refresh()
End Sub
Sub SimpleSquareAnywhere()
Dim Length As Double
Length = InputBox("Enter Length", "Square Length", 10)
On Error Resume Next
Dim P As Esprit.Point
Set P = Nothing
' this form of GetPoint will prompt the user to pick a point from the graphics
Set P = Document.GetPoint("Pick Corner Point for Square")
If P Is Nothing Then
Exit Sub
End If
' this time make S an array to keep track of all four segments
Dim S(1 To 4) As Esprit.Segment
Set S(1) = Document.Segments.Add(P, Document.GetPoint(P.X, P.Y + Length, 0))
Set S(2) = Document.Segments.Add(Document.GetPoint(P.X, P.Y + Length, 0), Document.GetPoint(P.X + Length, P.Y + Length, 0))
Set S(3) = Document.Segments.Add(Document.GetPoint(P.X + Length, P.Y + Length, 0), Document.GetPoint(P.X + Length, P.Y, 0))
Set S(4) = Document.Segments.Add(Document.GetPoint(P.X + Length, P.Y, 0), P)
Document.Refresh
End Sub

Do a Fit in ESPRIT EDGE after running the code and you will see the newly created segments in the graphics with one corner on the selected point.
