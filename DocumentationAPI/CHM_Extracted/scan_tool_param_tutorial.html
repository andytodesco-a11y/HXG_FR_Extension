Scan Tool Parameters

The Tutorial: Scan Tools used the generic Tool object to look at tools in the Tools collection. Each tool has a ToolStyle property that can be tested in order to cast that tool to a more specific type of object and gain access to its individual properties. Setting up the code to do that, however, can be a daunting task.

The Tool object itself is actually a special type of an even more generic object called Technology. Every Technology object is simply made up of a set of parameters. You can loop through all of the parameters without knowing exactly their names and get a Parameter object which has a Value and a Name properties.

Define additional tools or open other ESPRIT EDGE files that have other types of tools defined and look at the results of this code. You should notice that all properties and their values are displayed for each tool. E.g. ToolAngle and ToolDiameter are displayed for all of those tools that have those values.

public void ScanAllToolParameters()
{
_application.EventWindow.Clear();
_application.EventWindow.Visible = true;
var tools = Document.Tools as EspritTools.Tools;
foreach (EspritTechnology.ITechnology technology in tools)
{
foreach (EspritTechnology.Parameter parameter in technology)
{
_application.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ScanToolParametersTutorial", $"{parameter.Name} = {parameter.Value}");
}
}
}
Public Sub ScanAllToolParameters()
' prepare the output window for display
_application.EventWindow.Clear()
_application.EventWindow.Visible = True
' instead of declaring a generic Tool, use the generic Technology object
For Each toolTech As EspritTechnology.Technology In Document.Tools
' loop through all of the parameters on that technology
For Each param As EspritTechnology.Parameter In toolTech
' Param.Value should not be an Object
On Error Resume Next
_application.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ScanToolParametersTutorial", param.Name & " = " & param.Value)
Next
Next
End Sub
Sub ScanAllToolParameters()
' prepare the output window for display
Application.EventWindow.Clear
Application.EventWindow.Visible = True
' instead of declaring a generic Tool, use the generic Technology object
Dim ToolTech As EspritTechnology.Technology
For Each ToolTech In Document.Tools
' loop through all of the parameters on that technology
Dim Param As EspritTechnology.Parameter
For Each Param In ToolTech
' Param.Value should not be an Object
On Error Resume Next
Call Application.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ScanToolParametersTutorial", Param.Name & " = " & Param.Value)
Next
Next
End Sub
