Inserting Points into a FeaturePtop

This tutorial shows how to insert a point to a FeaturePtop. The code prompts the user to pick the FeaturePtop, pick the insertion location, and then the Point to insert, and than it inserts the selected point to a chosen locsation.

Now for the insertion location, there are several possibilities. For example, we could essentially Smash the FeaturePtop (inside of a SelectionSet), so that points were created at each location, then expect the user to pick one of those points. This may not be a good solution, though, because the user may have already had points there. As an alternative, we could use Grid mode, and expect the user to pick a grid point close to the location. Then we can just determine what location the user picked closest too.

Note that while FeaturePtop objects have Add, Count, and Item, they do not have Remove or Insert. So we cannot modify the end of the original FeaturePtop. Instead what we must do is create a new FeaturePtop which includes the point to be inserted, and remove the original.

public void InsertPointIntoFeaturePtop()
{
Esprit.FeaturePtop featurePtop = null;
try
{
featurePtop = Document.GetAnyElement("Pick Feature Ptop", EspritConstants.espGraphicObjectType.espFeaturePtop) as Esprit.FeaturePtop;
}
catch (System.Runtime.InteropServices.COMException)
{
// Element was not set
return;
}
if (featurePtop == null)
{
return;
}
var previousGridMode = _application.Configuration.EnableGridMode;
_application.Configuration.EnableGridMode = true;
Esprit.Point insertionPoint = null;
Esprit.Point point = null;
try
{
insertionPoint = Document.GetPoint("Pick Insertion Location");
}
catch (System.Runtime.InteropServices.COMException)
{
// Element was not set
return;
}
_application.Configuration.EnableGridMode = previousGridMode;
try
{
point = Document.GetPoint("Pick Point To Add");
}
catch (System.Runtime.InteropServices.COMException)
{
// Element was not set
return;
}
if (insertionPoint == null || point == null)
{
return;
}
var shortestDistance = 0.0;
var nearestItemIndex = 0;
for (var i = 1; i<= featurePtop.Count; i++)
{
var distance = Geometry.Distance(featurePtop.Item[i] as Esprit.Point, insertionPoint);
if (i == 1)
{
shortestDistance = distance;
nearestItemIndex = i;
}
else
{
if (distance < shortestDistance)
{
shortestDistance = distance;
nearestItemIndex = i;
}
}
}
Esprit.FeaturePtop secondFeaturePtop = null;
if (nearestItemIndex == 1)
{
secondFeaturePtop = Document.FeaturePtops.Add(point);
secondFeaturePtop.Add(featurePtop.Item[1] as Esprit.Point);
}
else if (nearestItemIndex > 1)
{
secondFeaturePtop = Document.FeaturePtops.Add(featurePtop.Item[1] as Esprit.Point);
}
else
{
return;
}
for (var i = 2; i <= featurePtop.Count; i++)
{
if (i == nearestItemIndex)
{
secondFeaturePtop.Add(point);
}
secondFeaturePtop.Add(featurePtop.Item[i]);
}
Document.FeaturePtops.Remove(featurePtop.Key);
Document.Refresh();
}
The following function is used to calculate distance between two points:
public static double Distance(Point point1, Point point2 = null)
{
var x2 = point2?.X ?? 0;
var y2 = point2?.Y ?? 0;
var z2 = point2?.Z ?? 0;
return Distance(point1.X, point1.Y, point1.Z, x2, y2, z2);
}
public static double Distance(double x1, double y1, double z1, double x2, double y2, double z2)
{
var dx = x2 - x1;
var dy = y2 - y1;
var dz = z2 - z1;
return Math.Sqrt(dx * dx + dy * dy + dz * dz);
}

Public Function Distance(Point1 As Esprit.Point, Point2 As Esprit.Point) As Double
Dim DeltaX As Double, DeltaY As Double, DeltaZ As Double
DeltaX = Point2.X - Point1.X
DeltaY = Point2.Y - Point1.Y
DeltaZ = Point2.Z - Point1.Z
Distance = Math.Sqrt(DeltaX ^ 2 + DeltaY ^ 2 + DeltaZ ^ 2)
End Function
Sub InsertPointIntoFeaturePtop()
Dim FP(0 To 1) As Esprit.FeaturePtop
On Error Resume Next
Set FP(0) = Document.GetAnyElement("Pick Feature Ptop", espFeaturePtop)
If FP Is Nothing Then Exit Sub ' in case user presses escape
Dim InsertionPoint As Esprit.Point, PointToAdd As Esprit.Point
' use grid mode to pick the insertion location, remember the previous state
Dim PreviousGridMode As Boolean
PreviousGridMode = Application.Configuration.EnableGridMode
Application.Configuration.EnableGridMode = True
Set InsertionPoint = Document.GetPoint("Pick Insertion Location")
If InsertionPoint Is Nothing Then Exit Sub ' in case user presses escape
Application.Configuration.EnableGridMode = PreviousGridMode
Set PointToAdd = Document.GetPoint("Pick Point To Add")
If PointToAdd Is Nothing Then Exit Sub ' in case user presses escape
Dim Distance2Points As Double, ShortestDistance As Double, NearestItem As Long
' loop through all of the points in the chosen feature ptop
Dim PointNumber As Long
For PointNumber = 1 To FP(0).Count
' find the nearest item to the chosen insertion location
Distance2Points = Distance(FP(0).Item(PointNumber), InsertionPoint)
If PointNumber = 1 Then
ShortestDistance = Distance2Points
NearestItem = 1
Else
If Distance2Points < ShortestDistance Then
ShortestDistance = Distance2Points
NearestItem = PointNumber
End If
End If
Next
' create a new feature ptop having the same attributes as the original
If NearestItem = 1 Then
' start with new point
Set FP(1) = Document.FeaturePtops.Add(PointToAdd)
Call FP(1).Add(FP(0).Item(1))
Else
Set FP(1) = Document.FeaturePtops.Add(FP(0).Item(1))
End If
FP(1).Color = FP(0).Color
FP(1).LineType = FP(0).LineType
FP(1).LineWeight = FP(0).LineWeight
' loop through the remaining points in the original, transfer them to the new one
For PointNumber = 2 To FP(0).Count
If PointNumber = NearestItem Then
' insert the new point in the appropriate location
Call FP(1).Add(PointToAdd)
End If
Call FP(1).Add(FP(0).Item(PointNumber))
Next
' remove the original feature ptop
Call Document.FeaturePtops.Remove(FP(0).Key)
Document.Refresh
End Sub
The following function is used to calculate distance between two points:
Public Function Distance(Point1 As Esprit.Point, Point2 As Esprit.Point) As Double
Dim DeltaX As Double, DeltaY As Double, DeltaZ As Double
DeltaX = Point2.X - Point1.X
DeltaY = Point2.Y - Point1.Y
DeltaZ = Point2.Z - Point1.Z
Distance = Sqr(DeltaX ^ 2 + DeltaY ^ 2 + DeltaZ ^ 2)
End Function

Before runing the code make sure you have one or more FeaturePtops in your ESPRIT EDGE file. Pick a FeaturePtop, then the insertion location, and a point to add. You should see the point to be added placed ahead of the insertion location. The reason for this is that it allows you to add a point prior to the start point. If you want to add a point after the end point, then you should simply re-open the FeaturePtop and pick the point to add.
