How to create ESPRIT EDGE C# tutorials extension

  1. Copy the folder with C# extension source files from "C:\Program Files\Hexagon\ESPRIT EDGE\Help\1033\API Tutorial Samples" to the appropriate place on your computer.
  2. Open and build the TutorialCSharp.sln with VS2017 or VS2019 or VS2022.
  3. After the successful build, the extension will be copied to "C:\Users\Public\Documents\Hexagon\ESPRIT EDGE\Data\Extensions\TutorialCSharp".




How to enable ESPRIT EDGE C# tutorials extension

  1. All C# tutorials mentioned in the help are installed with ESPRIT EDGE as C# extension. Check extension status through File->Extensions and enable it .
 
  1. Once enabled, you will see Tutorials tab on the ribbon with C# Tutorials under it.
 




Executing tutorial code samples

ESPRIT EDGE C# extension allows to execute the tutorials code samples. Click on Tutorial->C# Tutorials on the ribbon to see the available tutorials. To launch the code, select a tutorial in the tree and press the Execute button.
