Square FeatureChain

In this tutorial you will prompt the user to select a point and then create four segments to form a square with the chosen point as one of the corners. A FeatureChain will then be created around the square. Like the Square FeaturePtop Tutorial, this tutorial adds a feature onto geometry that was created in the previous tutorial. This time a FeatureChain is added to the FeatureChains collection. Note once again that the FeatureChain itself also has an Add method, but this Add method does not create objects, it only associates them with the FeatureChain, so there are differences in the syntax. This time a loop is used to add the segments to the FeatureChain.

Before running this code, create some points in ESPRIT EDGE. You will then be able to pick those points as the corner of the square feature. You can also use GRID or SNAP mode while running the code instead of creating points first.

public void SquareFeatureChain()
{
var length = 10.0;
if (RequestUserInput("Enter Length", "Square Length", ref length))
{
Esprit.Point point = null;
try
{
point = Document.GetPoint("Pick Corner Point for Square");
}
catch (System.Runtime.InteropServices.COMException)
{
// Element was not set
return;
}
if (point == null)
{
return;
}
var segments = new List<Esprit.Segment>
{
Document.Segments.Add(point, Document.GetPoint(point.X, point.Y + length, 0)),
Document.Segments.Add(Document.GetPoint(point.X, point.Y + length, 0), Document.GetPoint(point.X + length, point.Y + length, 0)),
Document.Segments.Add(Document.GetPoint(point.X + length, point.Y + length, 0), Document.GetPoint(point.X + length, point.Y, 0)),
Document.Segments.Add(Document.GetPoint(point.X + length, point.Y, 0), point)
};
var featureChain = Document.FeatureChains.Add(point);
featureChain.Color = ColorHelper.ColorToUInt(System.Drawing.Color.Blue);
foreach (var segment in segments)
{
featureChain.Add(segment);
}
Document.Refresh();
}
}
The following function is used to convert the color value:
public static uint ColorToUInt(System.Drawing.Color color)
{

Public Sub SquareFeatureChain()
Dim length As Double = 10.0
If Not RequestUserInput("Enter Length", "Square Length", length) Then
Return
End If
Dim point As Esprit.Point = Nothing
Try
point = Document.GetPoint("Pick Corner Point for Square")
Catch ex As System.Runtime.InteropServices.COMException
' Point was not set
Return
End Try
If point Is Nothing Then
Return
End If
Dim segments(0 To 3) As Esprit.Segment
segments(0) = Document.Segments.Add(point, Document.GetPoint(point.X, point.Y + length, 0))
segments(1) = Document.Segments.Add(Document.GetPoint(point.X, point.Y + length, 0), Document.GetPoint(point.X + length, point.Y + length, 0))
segments(2) = Document.Segments.Add(Document.GetPoint(point.X + length, point.Y + length, 0), Document.GetPoint(point.X + length, point.Y, 0))
segments(3) = Document.Segments.Add(Document.GetPoint(point.X + length, point.Y, 0), point)
Dim featureChain As Esprit.FeatureChain = Document.FeatureChains.Add(point)
featureChain.Color = ColorToUInt(System.Drawing.Color.Blue)
For i = 0 To 3
Call featureChain.Add(segments(i))
Next
End Sub
The following function is used to convert the color value:
Public Function ColorToUInt(color As System.Drawing.Color) As UInteger
Dim a As UInteger = color.A
Dim r As UInteger = color.R
Dim g As UInteger = color.G
Dim b As UInteger = color.B
Dim resuls As UInteger = ((a << 24) Or (b << 16) Or (g << 8) Or (r << 0))
Return resuls
End Function

Sub SquareFeatureChain()
Dim Length As Double
Length = InputBox("Enter Length", "Square Length", 10)
On Error Resume Next
Dim P As Esprit.Point
Set P = Nothing
' this form of GetPoint will prompt the user to pick a point from the graphics
Set P = Document.GetPoint("Pick Corner Point for Square")
If P Is Nothing Then
Exit Sub
End If
' this time make S an array to keep track of all four segments
Dim S(1 To 4) As Esprit.Segment
Set S(1) = Document.Segments.Add(P, Document.GetPoint(P.X, P.Y + Length, 0))
Set S(2) = Document.Segments.Add(Document.GetPoint(P.X, P.Y + Length, 0), Document.GetPoint(P.X + Length, P.Y + Length, 0))
Set S(3) = Document.Segments.Add(Document.GetPoint(P.X + Length, P.Y + Length, 0), Document.GetPoint(P.X + Length, P.Y, 0))
Set S(4) = Document.Segments.Add(Document.GetPoint(P.X + Length, P.Y, 0), P)
Dim FC As Esprit.FeatureChain
' creating new FeatureChain, so here use Set = syntax
Set FC = Document.FeatureChains.Add(P)
FC.Color = vbBlue
For I = 1 To 4
' adding existing objects to existing FeatureChain, so here use Call syntax
' note how S(I) is used to refer to a segment from the array
Call FC.Add(S(I))
Next
Document.Refresh
End Sub

Do a Fit in ESPRIT EDGE after running the code and you will see the newly created segments and the feature in the graphics with one corner on the selected point.
