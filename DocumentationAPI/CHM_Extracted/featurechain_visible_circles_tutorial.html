FeatureChain Visible Circles

This tutorial will loop through all of the circles in the Document and create features on all of the visible ones. These features will be created on the same Layer as the original Circle and will start at an angle entered by the user.

C#
VB.NET
VBA
public void FeatureChainVisibleCircles()
{
double angleDegree = 90.0;
if (!RequestUserInput("Enter Starting Angle", "Starting Angle", ref angleDegree))
{
return;
}
var angleRad = angleDegree * Math.PI / 180;
var previousActiveColor = Document.ActiveColor;
Document.ActiveColor = ColorHelper.ColorToUInt(Color.Blue);
foreach (Esprit.Circle circle in Document.Circles)
{
if (circle.Layer.Visible)
{
var featureChain = Document.FeatureChains.Add(
Document.GetPoint(circle.X + circle.Radius * Math.Cos(angleRad), circle.Y + circle.Radius * Math.Sin(angleRad), circle.Z));
featureChain.Layer = circle.Layer;
featureChain.Add(Document.GetArc(circle.CenterPoint, circle.Radius, angleRad, 2 * Math.PI + angleRad));
}
}
Document.ActiveColor = previousActiveColor;
WindowHelper.FitAllWindows(Document.Windows);
}
The following function is used to convert the color value:
public static uint ColorToUInt(System.Drawing.Color color)
{
var value = (uint)((color.A << 24) | (color.B << 16) |
(color.G << 8) | color.R);
return value;
}

The following function is used to fit all windows:
public static void FitAllWindows(Esprit.Windows windows)
{
foreach (Esprit.Window window in windows)
{
window.Fit();
window.Refresh();
}
}

Public Sub FeatureChainVisibleCircles()
Dim angle As Double = 90
If Not RequestUserInput("Enter Starting Angle", "Starting Angle", angle) Then
Return
End If
angle = angle * Math.PI / 180 'convert to Radians
' remember current settings
Dim previousActiveColor As Long = Document.ActiveColor
Document.ActiveColor = ColorToUInt(System.Drawing.Color.Blue)
' loop through all the circles
For Each circle As Esprit.Circle In Document.Circles
If circle.Layer.Visible Then
' create the feature chain
Dim featureChain As Esprit.FeatureChain = Document.FeatureChains.Add(Document.GetPoint(circle.X + circle.Radius * Math.Cos(angle), circle.Y + circle.Radius * Math.Sin(angle), circle.Z))
featureChain.Layer = circle.Layer
' because you cannot add a circle to a FeatureChain, make temporary arc instead
Call featureChain.Add(Document.GetArc(circle.CenterPoint, circle.Radius, angle, 2 * Math.PI + angle))
End If
Next
' reset to original value
Document.ActiveColor = previousActiveColor
Call FitAllWindows(Document.Windows)
End Sub
The following function is used to convert the color value:
Public Function ColorToUInt(color As System.Drawing.Color) As UInteger
Dim a As UInteger = color.A
Dim r As UInteger = color.R
Dim g As UInteger = color.G
Dim b As UInteger = color.B
Dim resuls As UInteger = ((a << 24) Or (b << 16) Or (g << 8) Or (r << 0))
Return resuls
End Function

Public Sub FitAllWindows(Windows As Esprit.Windows)
For Each window As Esprit.Window In Windows
window.Fit()
' note that Fit just sizes the window to fit the graphics
' it does not redraw the graphics, so we still need a Refresh
window.Refresh()
Next
End Sub

Before running this code, create several layers in ESPRIT EDGE and create some circles on each layer or load in a file that contains a lot of circles on different layers. Turn some of the layers off and leave some of the layers on. After running the code and you will see the newly created features in the graphics for every visible circle in the document. Turn layers on and off to confirm this. Notice that each feature is on the same layer as the circle from which it was created.
